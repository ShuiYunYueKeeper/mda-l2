## 录屏建议

| 场景 | 建议时长 | 要点 |
|------|----------|------|
| 源码编辑保存 | 15–30s | 编辑 → 实时预览 → `Ctrl+S` → dirty 标记消失 |
| 坏批注提示 | 10–20s | 编辑栏删掉 `]` → 预览不泄漏 → 保存弹窗提示 |
| 图片/流程图缩放 | 15–30s | 点击放大 → 滚轮缩放 → 拖拽平移 → 双击复位 |

可合并为单个 `gui-features-demo.gif` 替代多张静图。

## 与工作流的关系

3. 用户补充素材后，再更新 `README.md` 引用（可单独 `docs:` commit）。
