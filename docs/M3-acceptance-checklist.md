# M3 编辑 & 预览增强 — 实机验收清单

> 对应 P3 **M3-8**；验收标准见 [`P0-requirements-v2-commercial.md`](P0-requirements-v2-commercial.md) AC-1、AC-2、AC-2b。

```bash
npm run build && npm run gui
```

推荐测试文件：`samples/demo.md`（多级标题 + Mermaid）

---

## AC-1（预览·同步滚动 + 大纲）

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 打开 `samples/demo.md`，展开编辑栏（`Ctrl+E`） | 三栏可见 |
| 2 | 滚动**预览区** | 编辑区按比例跟随滚动 |
| 3 | 滚动**编辑区** | 预览区按比例跟随滚动 |
| 4 | 预览上方「大纲」面板 | 显示文档标题层级 |
| 5 | 点击大纲某标题 | 编辑区跳转到对应源码行 |

---

## AC-2（查找替换）

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | `Ctrl+F`，输入关键词 | 顶栏显示匹配数，当前匹配高亮 |
| 2 | `Enter` / ↓ | 跳转到下一处匹配 |
| 3 | `Ctrl+H`，输入替换文本 | 显示替换行 |
| 4 | 「替换」或「全部替换」 | 内容更新，可保存 |

大文件可用 `samples/perf-mixed.md` 或自行准备 5000+ 行文件。

---

## AC-2b（辅助编辑）

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 选中 `hello`，`Ctrl+B` | 变为 `**hello**`，光标合理 |
| 2 | 光标在普通行，`Ctrl+Shift+]` | 行首增加 `#` 升为标题 |
| 3 | 在代码围栏内尝试 `Ctrl+Shift+]` | **不修改**围栏内内容 |
