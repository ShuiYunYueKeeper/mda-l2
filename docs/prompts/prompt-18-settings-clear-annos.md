## Prompt：设置 + 筛选默认 + 清空

```
设置入口仅迁入自动保存；状态筛选默认只勾选 open；一键清空当前文件全部批注
（不做自定义批注级别；语言/开发者工具不进设置）
另增：设置项「记住界面习惯」「记住上次会话」（文件列表 + 当前文件；关则下次欢迎页）
```

### 实现摘要

- 视图 → 设置…（`Ctrl+,`）：自动保存四选一；「记住上次会话」「记住界面习惯」开关；点遮罩不关。
- 文件菜单移除「自动保存」submenu。
- `rememberSession`（`workspace-prefs.json`，默认开）：关则清除工作区根与最近文件、不再落盘；启动不自动打开 → 欢迎页。
- `mda-remember-layout`（默认开）：关则不读写布局习惯并清除已存键（批注栏/编辑栏/大纲/侧栏/宽度）。
- `filterStatus` 默认 `{ open: true, resolved: false, wontfix: false }`。
- core `clearAllAnnotations` + GUI「清空全部批注」确认按钮。
