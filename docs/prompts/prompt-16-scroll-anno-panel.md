## Prompt 41: 编辑区回车跳视口 / 批注首击定位

```
编辑区回车跳视口；点编辑同位置预览勿微调；批注首击定位不准；
批注栏记住展开；长批注可展开收起；新建后自动选中定位；批注弹窗点遮罩不关
```

AI 动作：

### 编辑 / 预览滚动
- Enter/Backspace/Delete：`pinEditorScroll` 钉住编辑区滚动（光标仍在原视口内则钉回）。
- 实时预览：`liveEdit` + `preserveScroll` + 冻结；勿两段式「先跳过 Mermaid 再补渲」（会导致预览先下后上闪烁）。
- 点编辑区定位预览：`onlyIfNeeded`；块顶部落在视口或有明显交集即视为已可见。

### 批注面板
- `mda-panel-visible` 记住展开/收起。
- 长批注（换行或 >72 字）：`is-collapsed` 三行钳制 +「展开/收起」（zh+en）。
- `locateAnnotationViews`：直接几何滚预览（避开 sync 锁串行丢第二次）+ 布局稳定后重试。
- 新建/编辑成功：`reloadFile({ selectAnnoId })`，Mermaid/色条就绪后 `selectAnnotation`。
- 增改批注弹窗：点遮罩不关闭（取消 / Esc 仍可关）。
